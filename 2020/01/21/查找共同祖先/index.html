在二叉树中两个不同节点的最近共同祖先节点
发表于 2020-01-21| 更新于 2020-01-29|C/C++
阅读量:

LCA（Last Common Ancestor）最近刷题遇到过这样一个题，在一个给定二叉树中，给出两个不同结点，要求求出这两个结点的最近祖先结点。

假设有一个二叉树（在markdown里我推不上去图片，只能用文字描述了）
A
B C
D E F G
A的左孩子是B，右孩子是C
B的左孩子是D，右孩子是E
C的左孩子是F，右孩子是G
首先结构体定义为
struct node{
int data;
node lchild,*rchild;
int layer;
};
LCA函数
node LCA(node* root,node* n1,node* n2){
if(root==NULL)return NULL;//如果当前结点为空结点，则返回NULL
if(root == n1 || root == n2 )return root;//如果当前节点为n1或n2，则返回当前结点

node* ll = LCA( root->lchild,n1,n2 );
node* rr = LCA( root->rchild,n1,n2 );

if(ll && rr)return root;//如果都不为空，则返回当前结点

if(ll==NULL)return rr;
else return ll;//如果一个为空，则返回另一个（即使另一个也为空）

}

这个函数用的递归思想。结合一个实例来描述一下这个函数调用过程。
例如，要求求出上面二叉树中结点D和F的最近祖先。
在main函数里
int main(){
node* lca = LCA(A,D,F);//此处用的伪代码

return 0;

}
函数LCA(A,D,F)中
root=A;
ll = LCA(B,D,F);//标记为1
rr = LCA(C,D,F);//标记为2

在函数1即LCA(B,D,F)中
  root=B;
  ll = LCA(D,D,F);//标记为3
  rr = LCA(E,D,F);//标记为4
  此时ll = D;rr = NULL;
  故函数1返回结点D

在函数2即LCA(C,D,F)中
    root=C;
    ll = LCA(F,D,F);//标记为5
    rr = LCA(G,D,F);//标记为6
    此时ll = F;rr = NULL;
    故函数2返回结点F

函数3，4，5，6很容易模拟运行过程    

现在我们再回头来看函数LCA(A,D,F)
   root=A;
   ll = LCA(B,D,F) = D;
   rr = LCA(C,D,F) = F;
   由于ll 和 rr都不为空，故返回结点root即A

  故D，F的最近祖先为A
